## <font color='red'> 1.1 K8s Cluster </font>

> Adapted from the [Kubernetes Tutorials](https://kubernetes.io/docs/tutorials/)  

fire up minikube:
```
minikube start
```

check minikube status:
```
minikube status
```
view addons:
```
minikube addons list
```

in a new terminal access dashboard:
```
minikube dashboard
```
